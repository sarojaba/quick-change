# Select

> Other names: Dropdown, Select input
>
> A form input used for selecting a value: when collapsed it shows the currently selected option and when expanded, it shows a scrollable list of predefined options for the user to choose from. - [Component Gallery](https://component.gallery/components/select/)

`Select`는 `Dropdown`과 생긴건 비슷하지만, 다음과 같은 차이점이 있다.
- 선택은 하지만, 액션은 수행하지 않는다.
- 주로 `Form`에서 사용된다.
