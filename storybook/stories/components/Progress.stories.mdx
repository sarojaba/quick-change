# Progress

> Other names: Progress
>
> A horizontal bar indicating the current completion status of a long-running task, usually updated continuously as the task progresses, instead of in discrete steps. - [Component Gallery](https://component.gallery/components/progress-bar/)

## Mode

Determinate
- 얼마나 진행되었는지 나타낼 때 사용합니다.
- `value`와 `max` 값을 요구합니다.

Indeterminate
- 얼마나 기다려야할지 모를때에 사용합니다.
- `value`와 `max` 값을 지정하지 않습니다.

주의 사항
- `NES`는 Indeterminate 모드를 지원하지 않습니다.
- `PaperCSS`는 Indeterminate 모드를 지원하지 않습니다.
