# Icon

> An icon is a graphic symbol designed to visually indicate the purpose of an interface element. - [Component Gallery](https://component.gallery/components/icon/)

- Ant Design은 [Ant Design Icons](https://ant.design/components/icon/)를 사용합니다.
- Atlaskit은 [Atlaskit Icon](https://atlaskit.atlassian.com/packages/design-system/icon)를 사용합니다.
- Bootstrap은 [Bootstrap Icons](https://icons.getbootstrap.com/)를 사용합니다.
- Bulma는 [Font Awesome](https://fontawesome.com/)을 사용합니다.
- Spectrum은 [Workflow icons](https://spectrum.adobe.com/page/icons/)를 사용합니다.
- Wired Elements는 [Material Icons](https://fonts.google.com/icons)를 사용합니다.
