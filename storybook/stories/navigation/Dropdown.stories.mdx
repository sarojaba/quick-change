# Dropdown

> Other names: Select menu
>
> A menu in which options are hidden by default, but can be shown by interacting with a button. Differs from a select in that it shows actions or navigation options and is not a form input. - [Component Gallery](https://component.gallery/components/dropdown-menu/)
